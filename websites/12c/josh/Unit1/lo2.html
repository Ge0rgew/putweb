2.1 - Types Of Software

Open Source Software

Source Code is the software code written by developers. When a piece of software is open source it means that users can view and edit the code. Linux and python are examples of open source software.

Benifits

Due to it being open source companys can restrict an customise to their needs.

Often low in price or free

Downsides

Code prone to errors

Security risk as editors can add malicous code

Closed Source Software

The code is restricted and only the developes can edit it. You buy the right to use but not own it.

Benifits

The code is well tested.

Support is easier as it has a list on how to fix errors.

Downsides

Customers rely on company to update the software.

Mainly paid software.

Off the shelf software

This is general use software like microsoft office and video games.

Benifits

Customers might already be familiar with the software as it is widespread used.

It can be cheaper as it is not customers for a user or company

Downsides

It can be expensive to purchases.

May contain features that you do not need.

Bespoke Software

bespoke Software is software that is custom made for a specific organsation or user. Schools will have bespoke software to track student records etc.

Benifits

The software is built to the companys need

Can be designed to the companys looks

Downsides

Expensive as it is curated to the companys needs.

Staff will require training as they will be unfamilliar.

As it is custom made origianl developers will need to maintain and update it.

Shareware

Shareware is software that allows the user to use the program for a limited time before making the buy it.

Anti-virus softare is an example as they give people a notification and then make the user pay after a month by putting pop ups in the corner.

Freeware

Freeware is free to download and to use. These are mainly closed-source software that earn money through averts or additional purchases.

Developers may release as freeware and then encourage the to purchase a full version with more features. Unlike shareware the user gets to use the free software for ever and have the choice to pay.

2.2 - Applications Software

Procutivity Software

This is general use software that is used to complete tasks accurately and efficiently.

Examples include Word Proccessors, Presentation Software, Web Browsers.

Applications like email applications are Benificial to organsation as they allow people to send files and photos to each other.

Development Tools

These are tools for programmers who are creating or modifying software.

An IDE (Intergrated Development Enviroment) is used to create and edit programs. They have a lot of tools built in like a translator to convert one programming language to another. A compiler to convert it into binary. A debugger to find issues in the code.

Business Software

This is specialist software curated for companys

Examples for this is CAD/CAM as it is used to design and construct products.

2.3 - Utillity Software

Anti-virus

Anti-virus software is designed to protect and remove any viruses that are on your computer. They are made to scan every file for malicous code or program to let the user know so that they can remove if from the system.

Firewall

A fireall manages incoming an outgoing trafic on a network. this protects the network from incoming attacks as it checks each data packet to see if it should be given access to the system/network.

Compression

Compression software is used to decrease the file size of a file. this allows more files to be saved onto one device. This allows for the file to transfered across a network quicker as the file will be smaller

Backing Up Data

A backup is a copy of data that can be used to restore a file if corrupt or lost.
